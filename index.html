Unidad 1

La sociedad digital: La sociedad digital es la comunidad de personas que usan la tecnología en su vida diaria, una comunidad conectada por lo digital, un ejemplo de esto puede ser el internet porque es un conjunto de personas conectadas, se comunican, realizan tareas de su vida cotidiana por medio de recursos digitales, como pedir comida o limpiar la casa con dispositivos inteligentes. Otro ejemplo de esto pueden ser las redes sociales.


Señales analógicas vs digitales: Las señales analógicas son señales de tipo continuo y están compuestas por varios valores que pueden cambiar, mientras que las señales digitales se usan para que los dispositivos digitales puedan funcionar y solo se componen de dos dígitos: 0 y 1.

La diferencia de estas señales se puede ver a través de sus ondas, como se puede ver a través de la siguiente imagen:

Las ondas de las señales analógicas pueden variar al ser valores continuos mientras que las señales digitales solo pueden variar entre 0 y 1.

Pirámide DIKW: Esta pirámide se puede describir como una jerarquía de datos que se van juntando para poder crear sabiduría a partir de los datos que son dados, la pirámide (de abajo para arriba) va en este orden:

  • Datos: Son los datos en bruto y no se puede sacar nada de estos en este estado.
  • Información: Son los datos de una forma más limpia y ordenada, ya tienen más sentido.
  • Conocimiento: Ya no son simplemente patrones, sino que se debe de analizar los nuevos datos para poder sacar una posible conclusión o solución en base a la información dada.
  • Sabiduría: Es cuando se toman decisiones debido a lo analizado en el paso anterior.

    Unidad 2

    Hardware y Software: La computadora se compone tanto de software como hardware, y estas se complementan entre sí para que la computadora pueda funcionar.

    El hardware es todo aquel dispositivo que se puede tocar, como el ratón, el monitor, una bocina. Mientras que el software es todo aquel que no se puede tocar como un sistema operativo o las aplicaciones que se encuentran adentro del dispositivo.

    Componentes de la placa base: La placa base es donde se colocan todos los componentes de la computadora para que esta pueda funcionar, entre estos componentes podemos encontrar:

  • Alimentación CPU y placa base: Es el componente que le da energía al CPU y otro que le da energía al resto de la placa.
  • Ranuras de memoria RAM: Donde se puede conectar memoria RAM adicional.
  • Pila BIOS: Pila que va en la placa madre y da energía a esta para que almacene la información de la BIOS (Basic Input Output System), y ser guardada en la memoria RAM.
  • M.2: Es una unidad de almacenamiento adicional que se le puede instalar a la computadora.
  • Socket: Es el lugar donde va el procesador y permite que procesos no relacionados puedan cambiar de información.
  • PCI: Existen varios tipos de PCI, estos sirven para conectar tarjetas gráficas o de sonido.
  • USB (Universal Serial Bus): Puerto donde se pueden conectar varios periféricos que usan el cable de USB.
  • Chipset: Controla el funcionamiento de los componentes de la placa base, a veces se separa en dos diferentes, uno norte y otro sur.
  • SATA: Conector donde se pueden conectar discos duros, o discos en formato DVD, CD o Blu Ray a la placa base.

    Unidad 3

    Redes: Las redes son un conjunto de dispositivos conectados entre sí a través de un medio que intercambian información y comparten recursos.

    El tipo de red depende de su magnitud, puede ser LAN (Local Area Network), WAN (Wide Area Network), el Intranet que es una red local para compartir info en una organización, la Extranet que es lo que conecta dos Intranets, o el Ethernet que es una red local por cable. La red inalámbrica o Wi-Fi es una red a la que no se necesita estar conectada con cable.

    También existe el VPN que es un servicio pagado que funciona como una red privada en la que los datos se trasladan de una forma más segura y cifrada para que pueda haber mayor privacidad para el usuario.

    Otros dispositivos: Existen otros dispositivos cuando se hablan de redes y ayudan a que estas puedan funcionar de una mejor manera, entre estos dispositivos se pueden mencionar:

  • Cables: Se usan varios tipos de cables para poder estar conectados a Internet, como el cable coaxial, el de fibra óptica, el de la línea telefónica,y por medio de satélite.
  • Switch: Este hace que los datos no se congestionen en una red, por lo que encuentra el mejor camino posible para transmitir la información.
  • Hub: Tiene la misma función que la del switch pero este manda la información a todas las computadoras, pero la información solo funciona parael destinatario, por lo que el ancho de banda puede disminuir.
  • Modem: Pasa la información de analógico a digital y viceversa, este es el que provee el servicio de Internet.
  • Router: Es el que da acceso y niega el acceso al Internet a los dispositivos electrónicos. También existe el modem-router.
  • Servidores: Estos son los que almacenan la información de muchos clientes y sus dispositivos.
